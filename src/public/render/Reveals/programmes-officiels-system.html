Le skill programmes-officiels

Un skill Claude pour rechercher et citer les programmes officiels

  • Recherche précise dans les Bulletins Officiels (BO)
  • Citations exactes des programmes collège et lycée
  • Référencement des compétences attendues
  • Liens au socle commun (collège)

Structure des programmes officiels

Les programmes sont organisés de manière hiérarchique

  • Thèmes : grandes catégories mathématiques (Nombres et calculs, Géométrie, etc.)
  • Attendus de fin d'année/cycle : objectifs généraux
  • Connaissances et compétences : détail des notions
  • Exemples d'activités : suggestions pédagogiques

calculer et résoudre des problèmes. Il comprend

de quotient et de proportion. »

\end{itemize}

L'agent programme-officiel-extractor

Un agent Claude spécialisé pour extraire et structurer les programmes depuis les PDF du BO

  • Analyse les PDF officiels du Bulletin Officiel
  • Extrait la structure (thèmes, attendus, compétences)
  • Découpe en compétences atomiques (insécables)
  • Génère des fichiers JSON structurés par niveau

Compétences atomiques

Principe : découper chaque compétence en unités insécables

Exemple : "Calculer avec des nombres décimaux"

Appuyez sur ↓ pour voir le découpage

Découpage atomique

Trop général

"Calculer avec des nombres décimaux"

Atomique

  • Additionner des nombres décimaux
  • Soustraire des nombres décimaux
  • Multiplier des nombres décimaux
  • Diviser des nombres décimaux

Chaque compétence atomique est la plus petite unité d'enseignement

Structure JSON générée

L'agent génère un JSON structuré pour chaque niveau

Appuyez sur ↓ pour voir la structure

Méthodologie de l'agent

1
Lire le PDF avec le skill pdf (page par page)
2
Identifier la structure (thèmes, attendus, compétences)
3
Découper en compétences atomiques
4
Générer des codes uniques (ex: cycle4_nombres_001)
5
Sauvegarder le fichier JSON structuré

Exemple de commande

Commande utilisateur à l'agent :

Analyse le PDF ".claude/skills/programmes-officiels/pdf/cycle4_v2.pdf"
pour le niveau CYCLE4 (5ème, 4ème, 3ème).

Sauvegarde le résultat dans
".claude/skills/programmes-officiels/data/cycle4.json"

avec les références BO depuis
".claude/skills/programmes-officiels/references_bo.json"

Résultat : Fichier JSON avec toutes les compétences atomiques du cycle 4

Conseils pratiques

  • Citations exactes
    pour documents officiels (inspecteurs)
  • Reformulation possible
    entre collègues (garder la référence)
  • Vérifier la date du BO
    les programmes évoluent
  • Exemples indicatifs
    les activités suggérées ne sont pas obligatoires
  • Relier au socle
    systématiquement pour le collège
  • Utiliser les codes
    pour traçabilité automatisée
