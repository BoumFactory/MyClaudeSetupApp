Transformer une ressource pédagogique (LaTeX, PDF, Markdown) en application web interactive autonome

Entrée

  • Corrections optionnelles

Sortie

  • Application web complète
  • Interface interactif
  • Prête à l'emploi
  • Distribution facile

Aucun code à écrire — juste fournir la ressource

Comment ça marche : processus automatisé

1
Analyse : Agent lit la ressource (LaTeX, PDF, MD)
2

Exemple 2 : Trigonométrie

  • Concept : Cercle trigonométrique interactif
  • Entrée : Angle en degrés ou radians
  • Animation : Position sur cercle, valeurs sin/cos/tan
  • Formules : \(\sin(\theta)\), \(\cos(\theta)\), \(\tan(\theta)\)

Élève bouge le curseur et observe la relation entre l'angle et les rapports trigonométriques

Technique : Construction

// Animation interactive : trigonométrie
class TrigonometryVisualizer {
  constructor(canvasId) {
    this.canvas = document.getElementById(canvasId);
    this.ctx = this.canvas.getContext('2d');
    this.angle = 0;
  }

  drawCircle() {
    // Dessiner le cercle unitaire
    this.ctx.beginPath();
    this.ctx.arc(this.canvas.width/2, this.canvas.height/2, 150, 0, 2*Math.PI);
    this.ctx.stroke();
  }

  updateAngle(theta) {
    this.angle = theta;
    this.redraw();
    // Mettre à jour sin(θ) et cos(θ) dynamiquement
    document.getElementById('sin-value').textContent = Math.sin(theta).toFixed(3);
    document.getElementById('cos-value').textContent = Math.cos(theta).toFixed(3);
  }
}

Le skill /create-app : Comment l'utiliser

Invocation simple : Uploader votre ressource et répondre à 3-4 questions

Syntaxe

/create-app
  --file mon_cours.tex
  --app-type auto
  --name "Second Degré"

Questions type

  1. Nom de l'application ?
  2. Type d'app (auto-détecté) ?
  3. Fonctionnalités optionnelles ?

Options de personnalisation :

  • Couleur du thème (pastel, sombre, vibrant)
  • Inclure/exclure certaines fonctionnalités
