Deux interfaces, un même moteur

Claude Desktop

  • Interface graphique (GUI)
  • 3 onglets intégrés
  • Idéal pour débuter
  • Pas de configuration
  • Multi-plateforme

Claude Code CLI

  • Interface terminal
  • Commandes personnalisées
  • Automation avancée
  • Scripts et workflows
  • Intégration DevOps
Point clé : Même intelligence Claude (GPT-4 level) dans les deux interfaces. Choisissez selon votre workflow, pas selon la performance.

Claude Desktop - Les 3 onglets

Chat

Conversation naturelle avec Claude. Partagez texte, code, images. L'interface standard.

Cowork (Collaboration)

Espace partagé pour travailler à plusieurs. Idéal pour pair programming ou co-création.

Code

Environnement spécialisé pour l'édition, le debugging et l'exécution de code. Intégration IDE.

Conseil : Commencez par Chat pour explorer, puis explorez Code et Cowork selon vos besoins.

Claude Code CLI - Installation

Prérequis

  • Git (Git for Windows sur Windows)
  • Compte Anthropic (console.anthropic.com)
  • Terminal : bash, zsh, PowerShell ou CMD

Installation native (recommandée)

macOS / Linux

curl -fsSL https://claude.ai/install.sh | bash

Windows (PowerShell)

irm https://claude.ai/install.ps1 | iex
Node.js n'est plus nécessaire. L'installeur natif est autonome, plus rapide, et se met à jour automatiquement.

Premier lancement

claude

La première exécution vous connecte à votre compte Anthropic via le navigateur.

Commandes de base

/help

Affiche la liste de toutes les commandes disponibles.

/compact

Résume la conversation pour réduire l'utilisation du contexte.

/clear

Efface l'historique et démarre une nouvelle conversation.

/exit

Quitte le programme proprement.

Utilisation : Tapez la commande dans la conversation (ex: /help) et appuyez sur Entrée.

Commandes personnalisées

Créez vos propres workflows automatisés via le fichier CLAUDE.md à la racine du projet.

Exemples intégrés

/createTex

Génère un fichier LaTeX structuré (article, présentation, etc.)

/createReveals

Crée une présentation reveal.js complète avec plan et slides.

Autocomplétion

Appuyez sur Tab pour afficher les commandes disponibles et compléter automatiquement.

Créez les vôtres : Éditez le fichier CLAUDE.md pour ajouter des commandes spécifiques à votre workflow.

Le fichier CLAUDE.md

Fichier de configuration qui influence le comportement de Claude dans tous vos projets.

Sans CLAUDE.md

$ claude-code > Qu'est-ce que vous voulez faire ? > Je dois créer un fichier LaTeX... > [Vous répétez chaque fois le contexte]

Avec CLAUDE.md

# Mon Workflow IA Tu es expert en création de documents LaTeX et présentations reveal.js. Commandes disponibles : - /createTex : Génère article LaTeX - /createReveals : Crée présentation reveal.js

Résultat : Claude comprend immédiatement votre contexte et met à disposition les commandes sans répétition.

Travailler avec des fichiers

Les trois opérations principales

Lire

Claude lit vos fichiers et comprend le code/contenu existant.

Utilisez @contexte ou partagez les chemins de fichiers.

Écrire

Claude crée de nouveaux fichiers (code, config, docs).

Validez les chemins et permissions avant génération.

Modifier

Claude met à jour vos fichiers (corrections, améliorations).

Les changements sont intelligents et minimaux (pas de réécriture complète).

Répertoire de travail

Claude travaille relativement à votre répertoire courant pwd. Toujours utiliser des chemins explicites.

La mémoire : gestion du contexte

Claude dispose d'une fenêtre de contexte limitée. Après ~100 messages, la conversation devient lente.

Quand utiliser /compact

  • Conversation dépasse 50-80 messages
  • Claude demande de résumer
  • Performance ralentit visiblement
  • Vous avez terminé une phase et commencez une nouvelle

Fonctionnement de /compact

$ /compact [Claude résume la conversation précédente] [Nouveau contexte compressé : ~10-15% de l'original] [Conversation continue avec mémoire compacte]
Conseil : Utilisez /compact régulièrement pour maintenir une session fluide et économique.
