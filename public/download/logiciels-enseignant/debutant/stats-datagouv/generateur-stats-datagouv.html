Generateur d'exercices Statistiques v0.3

Exercices de maths avec donnees reelles de data.gouv.fr — chaine de verite garantie (fichier brut + verification).

Prerequis : configurer le serveur MCP DataGouv

Ce prompt necessite un chatbot IA ayant acces aux serveurs MCP (Claude Code, Claude Desktop, Cursor...) avec le serveur MCP DataGouv configure.

Claude

Claude Code (terminal) :

claude mcp add --transport http datagouv https://mcp.data.gouv.fr/mcp

Claude Desktop (Settings > Connectors) :

{
  "mcpServers": {
    "datagouv": {
      "transport": "http",
      "url": "https://mcp.data.gouv.fr/mcp"
    }
  }
}

Gemini CLI

Gemini CLI (terminal) :

gemini mcp add --transport http datagouv https://mcp.data.gouv.fr/mcp

Configuration manuelle (settings.json) :

{
  "mcpServers": {
    "datagouv": {
      "httpUrl": "https://mcp.data.gouv.fr/mcp"
    }
  }
}

OpenAI

Responses API (dans le tableau tools) :

{
  "type": "mcp",
  "server_label": "datagouv",
  "server_url": "https://mcp.data.gouv.fr/mcp"
}

Source : github.com/datagouv/datagouv-mcp — Serveur MCP officiel (DINUM).

Chaine de verite (nouveau v0.3)

MCP DataGouv ──> URL fichier brut
│
Telechargement ──> raw_file.csv + source.json
│
Filtrage ──> extract.csv + extract.json + extract_params.json
│
Corrige ──> computed.json (chaque calcul declare)
│
verify.py ──> OK/FAIL par question

Le fichier brut est telecharge tel quel depuis data.gouv.fr. Tous les calculs du corrige sont verifiables contre les donnees source.
