# Assets Directory

Ce dossier est destiné à contenir des ressources (assets) utilisées par le skill agent-generator.

## Contenu

Les assets peuvent inclure :
- Templates d'agents pour des cas d'usage spécifiques
- Schemas JSON pour valider les structures d'agents
- Scripts utilitaires pour la génération d'agents

## Note

Actuellement, ce dossier ne contient pas d'assets car le skill agent-generator
utilise principalement des scripts et des références textuelles.

Les templates d'agents sont intégrés directement dans le script `init_agent.py`
pour une meilleure maintenabilité.
