## 2. Types de questions par domaine mathématique

### 2.1 Arithmétique et Nombres

**Ens. Scientifique :**
- Q1 : Calcul de pourcentage (opération basique)
- Q2 : Comparaison de nombres rationnels (fractions, décimaux)
- Q3 : Identification du plus grand nombre parmi puissances et décimaux
- Q5 : Calcul fractionnaire simple (tiers d'un quart)
- Q6 : Somme de décimaux et conversion

**Spécialité Maths :**
- Q1 : Inverse d'un nombre (manipulation algébrique)
- Q2 : Substitution dans formule avec fractions négatives
- Q5 : Calcul de probabilité complémentaire (utilisation de loi totale)

**Différence clé :** Ens. Scientifique privilégie les opérations élémentaires, Spécialité demande plus d'abstraction.

### 2.2 Pourcentages et Variations

**Ens. Scientifique :**
- Q1 : Calcul de pourcentage simple
- Q4 : Augmentations successives (10% + 10% ≠ 20%)
- Exercice 2, parties 2-5 : Pourcentages dans tableaux de contingence

**Spécialité Maths :**
- Q3 : Interprétation d'une multiplication par 0,975 (baisse de 2,5%)
- Q4 : Variations successives (+10% puis -10%)
- Exercice 2, parties 2-5 : Pourcentages avec fonction rationnelle

**Formulations types :**
- "L'opération qui permet de calculer X% de Y est..."
- "Le prix augmente de A%, puis augmente de B%. Après ces deux augmentations..."
- "Cela signifie que le prix de cet article a connu une baisse/augmentation de...%"
- "Quel est le pourcentage de X parmi Y ?"

### 2.3 Algèbre et Équations

**Ens. Scientifique :**
- Q11 : Développement d'identité remarquable (x-3)²
- Exercice 3 : Mise en équation d'un problème (tortue/escargot)

**Spécialité Maths :**
- Q2 : Substitution dans F = a + b/(c+d)
- Q6 : Équation fractionnaire : 1/x + 1/y = 1/u → u = xy/(x+y)
- Q9 : Identification de fonctions affines parmi non-affines
- Q12 : Moyenne pondérée (10×1 + 8×2 + 16x) / (1+2+x) = 15

**Différence clé :** Ens. Scientifique : identités remarquables et équations simples. Spécialité : manipulations algébriques abstraites.

### 2.4 Inéquations

**Ens. Scientifique :**
- Aucune QCM spécifique (mais notion implicite)

**Spécialité Maths :**
- Q7 : Résolution graphique (x² ≥ 10 → x ≤ -√10 ou x ≥ √10)
- Q11 : Étude du signe d'un produit (x × f(x) > 0)
- Exercice 2, partie 2 : Position courbe/axe via signe fonction

**Type de question :** "L'inéquation I est équivalente à..."

### 2.5 Géométrie et Vecteurs

**Ens. Scientifique :**
- Q9 : Équation de droite à partir de graphique

**Spécialité Maths :**
- Q8 : Équation de droite (4 formats différents)
- Exercice 1 : Vecteurs, produit scalaire, projection orthogonale, cercle
  - Calcul de coordonnées de vecteurs
  - Produit scalaire et longueurs
  - Équation cartésienne de droite
  - Équation du cercle et intersection

**Formulations types (Spécialité) :**
- "Déterminer les coordonnées du vecteur..."
- "En déduire le produit scalaire..."
- "Justifier qu'une équation du cercle est..."

### 2.6 Fonctions et Représentations graphiques

**Ens. Scientifique :**
- Q9 : Équation de droite à partir de graphique
- Q10 : Image d'un nombre par fonction f(x) = 7 - 2(x-3)²

**Spécialité Maths :**
- Q8 : Équation de droite (reconnaissance graphique)
- Q9 : Identification de fonctions affines (parmi f₁(x) = 2x(1-x), f₂(x) = (x²-1+√2)/2, f₃(x) = 0,7)
- Q10 : Reconnaissance de parabole (ouverture, décalage, formule)
- Exercice 2, partie 1 : Étude complète fonction g(x) = x² - 5x + 4
  - Signe (calcul des racines)
  - Suite de coefficients directeurs
- Exercice 2, partie 2 : Fonction rationnelle f(x) = x - 5 + 4/x
  - Dérivée et tableau de variations
  - Schéma de courbe

**Niveaux de difficulté :**
- Facile : "Quelle est l'image de 3 ?"
- Moyen : "Reconnaître une parabole à partir de graphique"
- Difficile : "Étudier signe, dérivée, variations"

### 2.7 Probabilités

**Ens. Scientifique :**
- Aucune QCM spécifique
- Exercice 2 : Probabilités dans tableau de contingence
  - "Lequel des deux a le plus de chance ?"

**Spécialité Maths :**
- Q5 : Probabilité manquante (loi totale)
  - 0,5 + 0,2 + x + 1/6 = 1 → x = ?
- Exercice 1 : Pas de probabilité directe

**Type de question :** "Déterminer la probabilité d'un événement parmi événements"

### 2.8 Statistiques

**Ens. Scientifique :**
- Q12 : Moyenne et médiane de deux séries
  - Série A : 1, 2, 3, 2
  - Série B : 0,5, 2, 100
  - "Une seule affirmation est exacte : A et B ont même..."

**Spécialité Maths :**
- Q12 : Moyenne pondérée avec inconnue
  - Notes : 10, 8, 16 avec coefficients 1, 2, x
  - Trouver x tel que m = 15

**Différence :** Ens. Scientifique compare deux séries. Spécialité résout équation sur moyenne.

### 2.9 Suites

**Ens. Scientifique :**
- Exercice 1 : Progression arithmétique (40 m²/semaine) et progression géométrique (20%/semaine)
  - "Quelle surface après 10 semaines ?"
  - "Est-il possible d'atteindre 580 m² ?"
  - "Au bout de combien de semaines ?"

**Spécialité Maths :**
- Exercice 2, partie 1.b : Suite de coefficients directeurs
  - aₙ = 2n - 4 (progression arithmétique)

**Type de question :** Modélisation de phénomène avec suite.

## 3. Formulations types par domaine

### Pourcentages
```
"L'opération qui permet de calculer X% de Y est..."
"Le prix augmente de A%, puis augmente de B%. Après ces deux augmentations..."
"Cela signifie que le prix de cet article a connu une baisse/augmentation de...%"
"Quel est le pourcentage de X parmi Y ?"
```

### Équations et Algèbre
```
"Lorsque a = 2, b = 3, c = 4, d = -1/4, la valeur de F est égale à..."
"On peut affirmer que... (option sans justification)"
"Déterminer en fonction de n la surface occupée après n semaines"
"Montrer que pour tout réel x de l'intervalle [0,5 ; 8], on a..."
```

### Géométrie analytique
```
"La seule équation pouvant correspondre à l'équation réduite de la droite D est..."
"Une équation de la droite D est..."
"Déterminer les coordonnées du vecteur..."
"Justifier qu'une équation du cercle est..."
```

### Fonctions
```
"L'image de X par la fonction f est égale à..."
"Parmi ces trois fonctions, celles qui sont affines sont..."
"Une seule des quatre fonctions ci-dessous est susceptible d'être représentée par..."
"Étudier le signe de la fonction g sur R"
"Montrer que la dérivée de f(x) est..."
```

### Inéquations
```
"L'inéquation I est équivalente à..."
"Résoudre x² ≥ 10"
"L'inéquation x×f(x) > 0 est vérifiée par..."
```

### Statistiques et Probabilités
```
"Une seule de ces affirmations est exacte : les deux séries ont..."
"Que doit valoir x pour que la moyenne = 15 ?"
"La probabilité d'obtenir chacune des faces est..."
"Lequel des deux a le plus de chance de remporter 1 euro ?"
```

### Suites et Modélisation
```
"Quelle sera la surface occupée par les nénuphars n semaines après l'anniversaire ?"
"Est-il possible qu'un dimanche, la surface soit égale à X m² ?"
"Au bout de combien de semaines, l'étang sera-t-il entièrement recouvert ?"
"On pourra s'aider du tableau ci-dessous"
```

## 4. Niveau de difficulté et progression

### QCM - Progression estimée de difficulté

**Ens. Scientifique :**
- Facile (Q1-Q6) : Calculs élémentaires, fractions, décimaux
- Moyen (Q7-Q10) : Puissances, conversions, équations de droite, images de fonction
- Difficile (Q11-Q12) : Identités remarquables, comparaison statistique

**Spécialité Maths :**
- Facile (Q1-Q4) : Inverses, substitutions, interprétations
- Moyen (Q5-Q9) : Probabilités, équations, graphiques, fonctions affines
- Difficile (Q10-Q12) : Paraboles, inéquations, moyennes pondérées, produit de signes

### Exercices - Structures de complexité

**Ens. Scientifique :**
1. Nénuphars : Progression arithmétique → géométrique → modélisation → représentation
2. Voitures : Tableau de contingence → pourcentages simples/croisés → probabilités
3. Course : Mise en équation unique

**Spécialité Maths :**
1. Géométrie : Vecteurs → produit scalaire → géométrie analytique → intersection
2. Analyse : Signe → suites → dérivée → variations → représentation

## 5. Format des réponses

### QCM
- **4 propositions systématiquement** (a, b, c, d)
- **Type de propositions :**
  - Nombres : décimaux, fractions, radicaux (√10), puissances (10^10)
  - Expressions : équations (y = x + 3), inéquations (x ≤ -√10)
  - Affirmations : "Les deux séries ont..."
  - Valeurs : pourcentages, longueurs

### Exercices
- **Parties identifiées :** 1.a, 1.b, 2.a, 2.b, 2.c, 3.a, 3.b, 3.c, 4
- **Types de réponses :**
  - Calcul numérique (résultat + unité)
  - Justification écrite (réponse par oui/non argumentée)
  - Équation/formule en fonction de n ou x
  - Tableau de variations
  - Schéma/graphique

## 7. Recommandations pour créer des QCM similaires

### Structure générale
- **12 questions systématiquement**, alternant difficultés
- **4 propositions (a, b, c, d)** avec exactement 1 bonne réponse
- **Topics couverts :** Algèbre (3-4 Q), Géométrie (2-3 Q), Fonctions (2-3 Q), Stats/Proba (1-2 Q), Autres (2-3 Q)

### Conception des distracteurs (mauvaises réponses)

**Ens. Scientifique :**
- Erreurs de calcul fréquentes (oubli parenthèses, priorités opératoires)
- Confusion entre concepts (moyenne/médiane, addition/multiplication)
- Pièges de formulation ("double de 5" vs "5 au carré")

**Spécialité Maths :**
- Erreurs algébriques (signe, parenthèses)
- Confusion entre concepts (fonction affine/quadratique)
- Mauvaise interprétation graphique (pente vs ordonnée)

### Gradation des exercices

**Modèle Ens. Scientifique (3 exercices) :**
1. **Suites + Modélisation** (4-5 questions progressives)
   - Calcul simple → condition d'existence → calcul avec table
2. **Tableaux de contingence** (6 questions : 1 lecture + 5 calculs)
3. **Mise en équation** (1 grande question ouverte)

**Modèle Spécialité Maths (2 exercices) :**
1. **Géométrie analytique** (3 questions progressives)
   - Calculs vectoriels → produit scalaire → équations analytiques
2. **Analyse complète** (3-4 questions progressives)
   - Étude algébrique → comportement avec suite → dérivation → variations

### Textes et contextes

**Contextes récurrents :**
- **Ens. Scientifique :** Situations réelles (étang, voitures, course)
- **Spécialité Maths :** Figures géométriques, fonctions mathématiques abstraites

**Longueur des énoncés :**
- QCM : 1-2 lignes max
- Exercices : 5-15 lignes avec support (tableau, graphique, figure)

### Aides fournies

**Ens. Scientifique :**
- Tableaux de valeurs (Ex1.q3)
- Graphiques sans échelle spécifiée
- Messages implicites : "On pourra s'aider du..."

**Spécialité Maths :**
- Décompositions de calcul intermédiaires (Ex1 : 2,5² = 6,25)
- Formules partiellement données
- Graphiques précis avec points

## 8. Synthèse comparative : points d'enseignement clés

| Aspect | Ens. Scientifique | Spécialité Maths |
|--------|-------------------|------------------|
| **Public cible** | Élèves sans spécialité maths | Élèves approfondissant maths |
| **Accent** | Applications concrètes, calcul | Abstraction, raisonnement |
| **Géométrie** | Équations de droites simples | Vecteurs, produit scalaire, cercles |
| **Algèbre** | Identités remarquables | Manipulations abstraites |
| **Analyse** | Images/équations | Dérivées, variations |
| **Justification** | Minoritaire en QCM | Absente en QCM |
| **Progressivité** | Linéaire (1→2→3) | Progressive dans chaque exercice |

## 9. Exemples concrets de formulations à réutiliser

### Question simple (Ens. Scientifique type)
```
L'opération qui permet de calculer 25% de 480 est :
a. 25 × 480 ÷ 100
b. 25 × 100 ÷ 480
c. ...
```

### Question avec graphique (Spécialité type)
```
On a représenté ci-contre une parabole P.
Une seule des quatre fonctions ci-dessous est susceptible
d'être représentée par la parabole P. Laquelle ?
a. x ↦ -x² + 10
b. x ↦ -x² + 10x
c. ...
```

### Exercice avec progression (Ens. Scientifique)
```
1.a. Quel pourcentage d'augmentation cela représente-t-il ?
b. Quelle est à présent la surface occupée ?

2.a. Quelle sera la surface après 10 semaines ?
b. Est-il possible qu'un dimanche la surface soit égale à 580 m² ?
c. Au bout de combien de semaines ?
```

### Exercice avec justifications (Spécialité)
```
a. Déterminer les coordonnées des vecteurs...
b. En déduire le produit scalaire...
c. Justifier qu'une équation du cercle est...
```

## 10. Conseils pratiques de rédaction

### Pour un QCM équilibré :
1. Alterner facile-moyen-difficile (F-M-F-M-D-F...)
2. Varier les domaines (ne pas 3 algèbre d'affilée)
3. Proposer 4 distracteurs plausibles
4. Vérifier unicité de la réponse

### Pour des exercices progressifs :
1. Partie A : Application directe (calcul)
2. Partie B : Adaptations ou justifications
3. Partie C : Synthèse ou cas limites
4. Inclure graphiques/tableaux de support

### Pour la clarté :
- Définir x = ? ou n = ? explicitement
- Préciser "en fonction de..." si réponse formule
- Donner tables de valeurs pour calculs exponentiques
- Utiliser notations cohérentes (f(x) vs y)
