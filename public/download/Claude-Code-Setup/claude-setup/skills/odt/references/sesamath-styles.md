# Styles Sesamath pour documents ODT educatifs

Ce fichier documente les styles personnalises extraits des cahiers Sesamath pour la creation de documents pedagogiques.

## Styles de paragraphe

### Styles de base

| Nom du style | Description | Usage |
|--------------|-------------|-------|
| `_Paragraphe` | Style de base pour le texte | Texte courant |
| `_Paragraphe_2pt` | Paragraphe avec espacement 2pt | Texte avec espacement reduit |
| `_Paragraphe_Bullet` | Paragraphe avec puce | Listes a puces |
| `_Paragraphe_Coche` | Paragraphe avec case a cocher | QCM, checklists |
| `_Paragraphe_Correction` | Paragraphe de correction | Corrections en rouge |

## Exemple d'utilisation XML

### Paragraphe avec style personnalise
```xml
<text:p text:style-name="_5f_Paragraphe">
  Texte avec le style _Paragraphe
</text:p>
```

### Exercice numerote
```xml
<text:p text:style-name="_5f_Titre_5f_Exercices_5f_avec_5f_Titre">
  Calculer les expressions suivantes :
</text:p>
```

### Zone de reponse eleve
```xml
<text:p text:style-name="_5f_Paragraphe_5f_Reponse_5f_Pointilles_5f_Grises">
  <text:s text:c="50"/>
</text:p>
```

**Note**: Dans les noms de styles XML, les underscores sont encodes en `_5f_` et les espaces en `_20_`.
