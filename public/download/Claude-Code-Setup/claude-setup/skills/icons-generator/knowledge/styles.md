# Styles visuels disponibles

Guide complet des styles pour g\u00e9n\u00e9ration d'images \u00e9ducatives.

## 1. minimal_color_print

**Usage principal** : Impressions quotidiennes, fiches d'exercices r\u00e9guli\u00e8res

### Caract\u00e9ristiques
- Palette limit\u00e9e : 2-3 couleurs maximum + noir
- Fond blanc ou tr\u00e8s clair
- Traits nets et \u00e9pais
- Contraste \u00e9lev\u00e9
- Pas de d\u00e9grad\u00e9s
- Style line art ou flat design

### Avantages
- \u00c9conomique en encre
- Impression rapide
- Lisibilit\u00e9 maximale
- Photocopies de qualit\u00e9

### Mots-cl\u00e9s recommand\u00e9s
- "simple line art"
- "minimal colors"
- "flat design"
- "high contrast"
- "white background"
- "clean lines"
- "economical printing style"

### Exemple de prompt
```
A mysterious locked door, simple line art, minimal colors (2-3 colors max),
high contrast, clean lines, white background, suitable for economical printing
```

---

## 2. moderate_color_classroom

**Usage principal** : Activit\u00e9s sp\u00e9ciales, projets, affichage en classe

### Caract\u00e9ristiques
- 4-6 couleurs vives
- D\u00e9tails moyens
- Style engageant et ludique
- \u00c9quilibre lisibilit\u00e9/attractivit\u00e9
- Adapt\u00e9 pour projection et impression

### Avantages
- Plus attractif visuellement
- Maintient l'attention des \u00e9l\u00e8ves
- Bon pour affichage num\u00e9rique
- Impression acceptable en couleur

### Mots-cl\u00e9s recommand\u00e9s
- "colorful illustration"
- "vibrant colors"
- "engaging and fun"
- "educational style"
- "medium detail"
- "classroom-friendly"

### Exemple de prompt
```
A treasure chest with golden coins, colorful illustration, 4-6 vibrant colors,
medium detail, engaging and fun, educational style, clear and readable
```

---

## 3. educational_illustration

**Usage principal** : Cours, sch\u00e9mas, fiches de synth\u00e8se

### Caract\u00e9ristiques
- Style didactique
- Clart\u00e9 maximale
- Possibilit\u00e9 d'annotations
- Simplification des formes
- Couleurs fonctionnelles

### Avantages
- Compr\u00e9hension facilitie
- Support p\u00e9dagogique efficace
- Utilisation scientifique/acad\u00e9mique
- Polyvalent

### Mots-cl\u00e9s recommand\u00e9s
- "educational diagram"
- "didactic style"
- "clear labels"
- "instructional design"
- "simple and understandable"
- "academic illustration"

### Exemple de prompt
```
Mathematical concept diagram showing vectors, clean educational diagram,
didactic style, clear labels possible, maximum clarity, instructional design
```

---

## 4. flat_design

**Usage principal** : Ressources modernes, num\u00e9rique

### Caract\u00e9ristiques
- Formes g\u00e9om\u00e9triques
- Composition simple
- Couleurs unies et vives
- Style minimaliste moderne
- Pas de textures

### Avantages
- Aspect contemporain
- Scalable (vectoriel friendly)
- Clair et efficace
- Adapt\u00e9 au num\u00e9rique

### Mots-cl\u00e9s recommand\u00e9s
- "flat design"
- "geometric shapes"
- "modern minimalist"
- "simple composition"
- "bold colors"
- "no textures"

### Exemple de prompt
```
Scientific laboratory equipment, flat design, geometric shapes,
simple composition, bold colors, modern minimalist style
```

---

## 5. cartoon_simple

**Usage principal** : Jeunes \u00e9l\u00e8ves (coll\u00e8ge), storytelling

### Caract\u00e9ristiques
- Personnages sympathiques
- Formes basiques
- Style approchable
- Ludique et fun
- 5 couleurs maximum

### Avantages
- Engageant pour jeunes
- Cr\u00e9e une connexion \u00e9motionnelle
- Facilite la narration
- M\u00e9morable

### Mots-cl\u00e9s recommand\u00e9s
- "simple cartoon style"
- "friendly characters"
- "basic shapes"
- "approachable and fun"
- "suitable for young students"

### Exemple de prompt
```
A curious student discovering a clue, simple cartoon style,
friendly characters, basic shapes, approachable and fun
```

## Conseils de s\u00e9lection

### Pour fiches d'exercices r\u00e9guli\u00e8res
→ **minimal_color_print**

### Pour escape games et activit\u00e9s th\u00e9matis\u00e9es
→ **moderate_color_classroom** ou **cartoon_simple**

### Pour sch\u00e9mas et explications
→ **educational_illustration**

### Pour ressources num\u00e9riques
→ **flat_design** ou **moderate_color_classroom**

### Pour jeunes \u00e9l\u00e8ves
→ **cartoon_simple**

### Pour lyc\u00e9e/sup\u00e9rieur
→ **educational_illustration** ou **flat_design**
